nodeJs全局变量：process

__dirname显示当前文件的目录路径（绝对路径）

__filename显示当前文件的路径（绝对路径）

####第三方模块

require()是引入模块的方法

moudle.exports是导出模块的方法

下载npm模块使用：npm install name

####npm的发布
1.符合规范的目录需求：

package.json

README.md(可选的)

2.npm 注册：npm adduser

3.初始化npm目录：npm init

4.目标模块上传：npm publish 目标模块删除（已上传）：npm unpublish
