面向对象基础

Veröffentlicht am 2016-05-31

什么是面向对象

面向对象是一种软件开发的编程方式，与面向过程是两种不同的编程方式

特点

面向过程：所有功能都是在需要使用的时候开发

面向对象：在正式开发之前，就先把所有需要使用的功能都开发好，并以属性或者方法的形式存放在一个对象中，在实际开发的时候根据需要来调用写好的功能

JavaScript是最不像面向对象的面向对象语言，因为JavaScript中没有类的概念，也不能像传统面向对象语言那样去实现封装、继承和多态。

面向对象模式

面向对象的工厂模式

function person(name,age){
    var obj={};
    obj.name=name;
    obj.age=age;
    obj.say=function(){
        alert('我叫'+this.name+'，我今年'+this.age+'岁。');
    };
    return obj;
}
var xiaoming=person('小明',18);
var xiaohong=person('小红',16);
xiaoming.say();
xiaohong.say();

1.工厂方式写法与面向过程写法相同，相对比较好了解

2.工厂方式的缺点在于所有属性和方法都会被重复创建

####面向对象的构造函数模式
function Person(name,age){
this.name = name;
this.age = age;
this.say = function(){
alert(‘我叫’+this.name+’,我今年’+this.age+’岁。’)；
};
}
var xiaoming = new Person(‘小明’,18);
var xiaohong = new Person(‘小红’,16);
xiaoming.say();
xiaohong.say();

构造函数：用来构建对象的函数

1.为了区别普通函数，构造函数的函数名首字母规定应该为大写。

2.构造函数必须使用new运算符来调用执行。

普通函数与构造函数中的this。

普通函数的this与调用时的上下文环境有关。

构造函数中的this为一个空的对象，并且会作为return的默认值。

JavaScript内置提供的构造函数

String() / Array() / Date() / Number() / Boolean() / Object() / Function() / RegExp()

构造函数+原型模式

function Person(name,age){
    this.name=name;
    this.age=age;
}
Person.prototype.say=function(){
    alert('我叫'+this.name+',我今年'+this.age+'岁。');
};
var xiaoming=new Person('小明',18);
var xioahong=new Person('小红',16);
xiaoming.say();
xiaohong.say();

原型是函数的一个特有属性。

1.原型本身是一个对象。

2.原型对象上的属性和方法可以被实例对象使用。

普通对象中的属性与原型中的属性的区别

1.普通对象的属性：只能在拥有属性的对象中使用。

2.原型对象的属性：所有通过同一个构造函数创建的对象中都可以使用。

原型链：原型链是指对象在访问属性或方法时的查找方式。

1.当访问一个对象的属性或方法时，会先在对象自身上查找属性或方法是否存在，如果存在就使用对象自身的属性或方法。如果不存在就去创建对象的构造函数的原型对象中查找，以此类推，直到找到为止。如果到顶层对象中还找不到，则返回undefined.

2.原型链最顶层为Object构造函数的prototype原型对象，给Object.prototype添加属性或方法可以被除null和undefined之外的所有数据类型对象使用。

constructor与instanceof

constructor属性(举例:nodeType)

通过constructor属性可以查看一个实例对象是由哪一个构造函数创建出来的。

  1. constructor是构造函数的prototype中的属性,它的值为构造函数本身

  2. constructor属性可以被修改

instanceof运算符

instanceof运算符用来查看一个实例对象是不是由某个构造函数创建出来的

面向对象版选项卡

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8" />
        <title></title>
        <style>
              #tab {width: 400px;border: 5px solid #333;background: #9cf;}
              #tab input {font-size: 30px;background: #fff;}
              #tab div {height: 200px;background: #eee;display: none;font-size: 80px;}
              #tab input.active {background: red;}
              #tab div.active {display: block;}
        </style>
    </head>
    <body> 
        <div id="tab">
             <input class="active" type="button" value="选项一" />
             <input type="button" value="选项二" />
             <input type="button" value="选项三" />
             <div class="active">0000001</div>
             <div>0000002</div>
             <div>0000003</div>
         </div>
         <script>
            function Tab(id){  
                  this.oTab = document.getElementById(id);
                  this.aBtn = this.oTab.getElementsByTagName('input');
                  this.aDiv = this.oTab.getElementsByTagName('div');
                  var _this = this;
                 for(var i=0; i<this.aBtn.length; i++){
                    this.aBtn[i].index = i;
                    this.aBtn[i].onclick = function(){
                          _this.click(this.index);         
                        };
                  }

            }

            Tab.prototype.click = function(index){
                  for(var i=0; i<this.aBtn.length; i++){
                    this.aBtn[i].className = '';
                    this.aDiv[i].className = '';
                  }
                  this.aBtn[index].className = 'active';
                  this.aDiv[index].className = 'active';
            };

            new Tab('tab'); 

        </script>
    </body>
</html>

将普通版改成面向对象版：

1.所有变量改成对象的属性

2.把所有的函数改成构造函数的原型上的方法

3.方法中不要有函数嵌套
