JavaScript语法特性面试题

Veröffentlicht am 2016-05-23

####1
function Foo() {
getName = function () { alert (1); };
return this;
}
Foo.getName = function () { alert (2);};
Foo.prototype.getName = function () { alert (3);};
var getName = function () { alert (4);};
function getName() { alert (5);}

//请写出以下输出结果
1.Foo.getName();
2.getName();
3.Foo().getName();
4.getName();
5.new Foo.getName();
6.new Foo().getName();
7.new new Foo().getName();

答案是：

1. 2 
2. 4
3. 1
4. 1
5. 2
6. 3
7. 3

详细解析
先将这几个函数按优先级排列

function Foo(){
    getName = function (){ alert (1); };
    return this;
};

function getName(){
    alert(5);
};

var getName;

Foo.getName = function () { 
    alert (2);
};

Foo.prototype.getName = function () { 
    alert (3);
};

getName = function() { 
    alert (5);
}

1.第一问的 Foo.getName 自然是访问Foo函数上存储的静态属性，自然是2

2.直接调用getName函数，而var getName = function(){ alert(4); }中的变量声明提升,
得出“var getName;getName=function(){ alert(4); }”,后面的覆盖前面的getName函数，故此题为4

3.先将Foo()函数运行，其中的return值为this，this指向window,所以第3题就等于window.getName(),Foo函数运行后其中的getName函数在全局中出现，放在最后，所以此题弹出值为1

4.直接调用getName函数，与上题相同，弹出1

5.new Foo.getName(),此处考察的是js运算符优先级的问题，.的运算级高于new操作，于是为new（Foo.getName）(),所以弹出2

6.new Foo().getName(),运算符优先级（）高于new，所以为（new.Foo()）.getName()
