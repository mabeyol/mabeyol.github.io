bootstrap应用

全局样式

  1. 移除了body的margin声明

  2. 设置body的背景色为白色

  3. 为排版设置了基本的字体、字号和行高

  4. 设置了全局连接颜色，且当连接处于悬浮 “:hover”状态时才会显示下滑线样式

标题样式

重置了h1-h6的默认样式

1.重新设置了margin-top和margin-bottom的值。h1-h3重置后的值都是20px；h4-h6充值后的值都是10px

2.所有标题的行高都是1.1（也就是font-size的1.1倍），而且文本颜色和字体都继承父元素的颜色和字体

3.固定不同级别标题字体大小：h1=36px，h2=30px，h3=24px，h4=18px，h5=14px和h6=12px

标题类名

标题类(.h1-.h6),向块级元素添加该类名后，可以让该块元素得到标题样式

副标题small

<h1>Bootstrap标题一<small>我是副标题</small></h1>

small标签可以配合标题标签使用，使small标签里的文字成为副标签

强调内容

1.强调标签：

①.lead 一般配合段落使用

②small标签

③加粗 strong标签 b标签

④斜体 em标签 i标签 cite标签

2.强调相关的类：

text-muted：提示，使用浅灰色（#999）
text-primary：主要，使用蓝色（#428bca）
text-success：成功，使用浅绿色(#3c763d)
text-info：通知信息，使用浅蓝色（#31708f）
text-warning：警告，使用黄色（#8a6d3b）
text-danger：危险，使用褐色（#a94442）

文本对齐风格

text-danger：危险，使用褐色（#a94442）
text-left：左对齐
text-center：居中对齐
text-right：右对齐
text-justify：两端对齐

列表

1.ul,ol样式清除:添加list-unstyled类

2.ul,ol水平列表:添加list-inline类

把垂直列表换成水平列表,而且去掉项目符号（编号）,保持水平显示

3.dl水平列表:添加dl-horizontal类

屏幕大于768px的时候,添加类名”.dl-horizontal”才具有水平定义列表的效果

代码风格

！！！不管使用哪种代码风格，在代码中碰到小于号(<)要使用硬编码”<”来替代,大于号(>)使用”>”来替代

1.code:一般针对于单个单词或单个句子的代码

2.pre:一般是针对于多行代码(也就是成块的代码)

图片样式

1.img-responsive:响应式图片，主要针对于响应式设计

2.img-rounded:圆角图片

3.img-circle:圆型图片

4.img-thumbnail:缩略图片

图标样式

1.任何元素都可以作为图标的容器，通常使用span标签做图标容器

2.bootstrap框架中为大家提供了近200个不同的icon图片，网址为http://getbootstrap.com/components/#glyphicons

3.bootstrap框架中是通过给元素加类名来实现的，格式为class:”glyphic 图标名”

CSS伪元素

CSS伪元素用于像某些选择器设置特殊效果

伪元素语法：selector::first-line{ background:red; }

常用的伪元素：::before , ::after , ::first-line , ::first-letter , ::selector

::before和::after伪元素

::before是在元素之前添加内容，::after是在元素之后添加内容

必须要添加content属性，无论伪元素是否有内容

搭建本地博客

  1. 安装hexo

  2. 创建一个文件夹，命名为myblog

  3. 在终端进入myblog文件夹，输入 hexo init 初始化文件夹，文件夹中文件如下：

4.在终端中输入 hexo new “我的第一篇博客” 创建新博客；删除博客直接在根目录下删除就可以

5.用submit或markdown对博客内容进行修改，然后在终端输入 hexo server ，打开命令行中的链接，就可查看本地博客myblog

使用github

1.打开浏览器，在地址栏输入github.com，进入github官网

2.在官网上注册一个github账号

3.进入账号，点击右上角的 “+”，找到 new repositories ,然后创建一个仓库

4.返回账户主页，点击你的头像，点击setting，在setting选项中点击SSH and GPG keys，然后点击 New ssh key;

5.在终端中输入 git clone 仓库地址
用来下载线上仓库地址

6.在终端中输入vim ~/.ssh/id_rsa.pub 会出现一段长代码，将代码复制进去ssh key中的key中（退出为shift+：，然后输入q再点击空格），点击创建ssh key

7.在终端中输入$ git config –global user.name “Your Name”和$ git config –global user.email “email@example.com”来配置你的账户名和邮箱

8.打开myblog下的_config.yml,输入信息如下：

10.在浏览器地址栏输入用户名.github.io进入线上博客
